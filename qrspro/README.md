# QSRPro Website

Professional static website for QSRPro - the troubleshooting platform for quick-service restaurants.

## Overview

QSRPro is a revolutionary troubleshooting platform designed specifically for quick-service restaurant equipment. This website serves as the landing page and early access portal for the platform.

## Usage

### Local Development
1. Open `index.html` in a web browser
2. All styling and scripts are self-contained
3. No build process required

### Deployment
1. Upload all files to web server
2. Ensure proper MIME types for CSS and JS
3. Configure SSL/TLS for HTTPS
4. Set up form submission backend (optional)

### Form Integration
The forms are currently set up to log submissions to the console. To integrate with a backend:

1. Update the form submission handlers in `script.js`
2. Send form data to your backend API
3. Implement email notifications
4. Store submissions in database
